A fully functional Uber-like taxi booking app built with React Native + Expo, featuring live GPS tracking, route drawing, dynamic fare calculation, ride simulation,
Features
Live GPS tracking

Map with pickup & drop markers

Route drawn between locations

Distance, time, and fare calculation

Ride types: Mini, Sedan, SUV

Driver movement simulation

Complete ride flow: Search → Confirm → Ride → Summary

APK build using Expo EAS

Tech Used
React Native (Expo)

React Navigation

React Native Maps

Expo Location

OpenStreetMap (OSRM)

Run way 
npm install
npx expo start
